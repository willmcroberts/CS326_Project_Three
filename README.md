# Sudoku and Map Solver
This program solves sudoku as well as map coloring puzzles
## Running the program
*The program should run the sudoku that is in the files by default.*  

To choose to switch the input, scroll down the glam `✨` emojis. There, you will be able to change the type of puzzle the program is solving, as well as the actual starting puzzle.
There is one sudoku and one map coloring problem already in the files, you can add more by creating a new file in the `puzzles` folder.  

### Ex.  
------------------------------------------
```
    problem = "sudoku"
    instance = "puzzles/sudoku1.json" 
    config = "mrv_fc"
```
------------------------------------------
```
    problem = "map"
    instance = "puzzles/australia.json" 
    config = "mrv_fc"
```
